# Project Overview
Wander AI is a locally orchestrated, multi-agent AI ecosystem designed for IDE-first workflows. It solves the friction of context-switching during "vibe coding" by bringing an entire agency of 16 specialized AI developers (e.g., Database Architect, DevOps Engineer, Frontend Developer) directly into the user's terminal and IDE.

# The Architecture (Monorepo)
The ecosystem ships as three complementary tools that share a single underlying agent registry:
1. **Web Chat UI:** A responsive, browser-based React interface with a sleek dark mode and interactive documentation hub.
2. **Node.js CLI (`wanderai`):** A terminal-first tool for executing tasks globally.
3. **MCP Server (`wanderai-mcp`):** A native Model Context Protocol integration that allows IDEs like Cursor, Windsurf, and Claude Desktop to delegate tasks directly to our specialized agents.

**Core Tech:** Node.js, Next.js/React, Tailwind CSS, `@modelcontextprotocol/sdk`.
**LLM Gateway:** Multi-provider API routing natively supporting OpenAI, OpenRouter, Google Gemini, and NVIDIA NIMs.
